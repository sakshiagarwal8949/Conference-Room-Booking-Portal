A clean, functional room booking portal with 3 modules — check availability, book a 
slot, and cancel bookings. Built solo in under 15 days using HTML, CSS & JavaScript.

This was my first real web project with an actual use case — managing conference room 
bookings for an organization. I built the entire thing solo during my internship at 
SKIT's Incubation Cell.

It has three core flows: checking if a room is free, booking it for a time slot, and 
cancelling an existing booking. Simple on the surface, but it taught me a lot about 
thinking through user flows and edge cases.

Built with: HTML5, CSS3, JavaScript
